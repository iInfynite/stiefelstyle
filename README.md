# StiefelStyle - Luxury Fashion Landing Page

[**Live Website**](https://stiefelstyle.com)

StiefelStyle is a high-end editorial landing page showcasing premium winter footwear. Built for the German luxury market, the platform combines classic elegance with modern web performance to deliver a boutique digital experience.

---

## 💼 Client Project
This platform was custom-developed specifically for a **German-based client** in the luxury retail sector. The project focused on high-impact visual storytelling, premium typography, and localized compliance (DSGVO) for the elite fashion market in the DACH region.

## 🛠️ Tech Stack
- **HTML5**: Semantic markup for premium content delivery and SEO.
- **Vanilla CSS**: Advanced styling using **Playfair Display** (Headings) and **Lato** (Body) font-families. Features include a custom gold-shimmer logo animation, elegant grid layouts, and sophisticated depth effects.
- **JavaScript**: Custom logic for a sticky header, smooth accordion FAQs, and localized legal compliance modals.

## 🌟 Key Features
- **Luxury Aesthetic**: A curated design system using gold accents and beige/nude color palettes to reflect modern sophistication.
- **Editorial Experience**: Large, high-quality imagery combined with elegant typography to guide the user through the winter collection.
- **Responsive Boutique**: A seamless experience across mobile and desktop, ensuring the luxury feel is never compromised.
- **Full Legal Compliance**: Integrated DSGVO-compliant disclaimer bars, cookie consent banners, and Impressum/Privacy systems.
